# Simulation

Benchmarking TriGer community detection on synthetic data with planted communities.

## Results

**[View Simulation 1 analysis](https://jy2941.github.io/TriGer/simulation/html/matrix_1_publish.html)**

## Files

| File | Description |
|------|-------------|
| `matrix_1_generate.m` | Generate synthetic 1000×1000 correlation matrix with two planted communities |
| `matrix_1_TGI.m` | Run TriGer across repeated samples and evaluate TPR/TNR |
| `matrix_1_publish.m` | Single-run walkthrough with figures |
| `matrix_1_benmark.R` | R benchmark comparing TriGer against alternative methods |
| `html/` | Published HTML output with figures |
